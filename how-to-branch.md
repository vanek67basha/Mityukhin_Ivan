# Как создать ветку
Чтобы создать ветку, необходимо ввести команду в терминале:
'git branch name'
где name - имя, создаваемой ветки.

# Как переключиться в ветку
Для переключения на другую ветку понадобится команда:
'git checkout name'
где name - имя ветки.

# Как отправить ветку на сервер
Отправляем ветку на сервер также, как отправляем в обычном случае. Единственное6 терминал предложит ввести нам строку:
'git push --set-upstream origin name'
где name - имя нашей ветки.
